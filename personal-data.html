Политика обработки персональных данных

Дата последнего обновления: 31 января 2025 г.

1. Введение

Настоящая Политика определяет порядок обработки персональных данных и меры по обеспечению их безопасности в сервисе HEY, STORE! (далее — «Оператор»).

2. Основные понятия

  • Персональные данные — любая информация, относящаяся к определённому физическому лицу
  • Обработка персональных данных — любое действие с персональными данными
  • Оператор — лицо, осуществляющее обработку персональных данных
  • Субъект персональных данных — физическое лицо, чьи данные обрабатываются

3. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Согласия субъекта персональных данных
  • Необходимости исполнения договора
  • Требований законодательства Российской Федерации

4. Перечень обрабатываемых данных

Оператор обрабатывает следующие категории персональных данных:

  • Контактные данные (адрес электронной почты)
  • Данные о заказах и транзакциях
  • Технические данные (IP-адрес, cookies, данные браузера)

5. Цели обработки

  • Идентификация пользователя для оказания услуг
  • Обработка и выполнение заказов
  • Связь с пользователем
  • Улучшение качества сервиса
  • Проведение статистических исследований

6. Сроки обработки и хранения

Персональные данные обрабатываются и хранятся в течение срока действия согласия субъекта, но не более 3 лет с момента последнего взаимодействия.

По истечении срока хранения данные уничтожаются.

7. Меры по защите данных

Оператор принимает следующие меры по защите персональных данных:

  • Использование защищённых протоколов передачи данных (HTTPS)
  • Ограничение доступа к персональным данным
  • Регулярный аудит систем безопасности
  • Обучение сотрудников правилам работы с персональными данными

8. Права субъекта данных

Субъект персональных данных имеет право:

  • Получить информацию об обработке своих данных
  • Требовать уточнения, блокирования или уничтожения данных
  • Отозвать согласие на обработку данных
  • Обжаловать действия Оператора в уполномоченный орган

9. Cookies и аналитика

Сайт использует файлы cookies для улучшения работы сервиса. Вы можете отключить cookies в настройках браузера.

10. Изменения в политике

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия всегда доступна на данной странице.

11. Контактная информация

По вопросам обработки персональных данных обращайтесь через Telegram.
